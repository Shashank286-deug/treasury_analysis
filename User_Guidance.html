Treasury & Investment Analysis User Guidance

Step-by-Step Guide

  • Click 'Guidance' button on the web app homepage to access this guide
  • Ensure an active internet connection
  • Prepare a CSV file with 'Date', 'Inflow', and 'Outflow' columns
  • Locate 'Upload Cash Flow Data (CSV)' section
  • Click 'Choose File' to select your CSV
  • Submit by clicking the upload area or pressing 'Enter'
  • Select 'Use Sample Data' checkbox if no CSV is available
  • Check 'Detected Columns' list after upload
  • Choose 'Date Column' from dropdown menu
  • Choose 'Inflow Column' from dropdown menu
  • Choose 'Outflow Column' from dropdown menu
  • Verify 'Standardized Data' table for correct mappings
  • Adjust 'Forecast Periods (Months)' slider (1-24 months)
  • Set 'Growth Rate' slider (0.0-0.1)
  • Check 'Use Current 10-Year Treasury Yield' or set 'Discount Rate' slider (0.05-0.15)
  • Enter 'Initial Investment' in number input
  • Adjust risk factors (Policy, Intermittency, Market, Operational) sliders for RARM
  • Review 'Cash Flow Forecast', 'DCF Valuation', 'RARM', and 'Sensitivity Analysis' results
  • Scroll to 'Visualizations' for 'Cash Flow Forecast' chart
  • View 'Sensitivity Analysis' heatmap below the chart
  • Go to 'Download Report' section
  • Click 'Download Report (Excel Only)' for Excel data
  • Click 'Download All Data (ZIP)' for ZIP with data and charts

Data Requirements

  • CSV file must include a 'Date' column in YYYY-MM-DD format
  • 'Inflow' column must contain numeric values representing cash inflows
  • 'Outflow' column must contain numeric values representing cash outflows
  • Ensure no blank rows or special characters in data columns
  • Sample data available if no CSV is uploaded
