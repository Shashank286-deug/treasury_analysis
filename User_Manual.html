Treasury & Investment Analysis User Manual

Getting Started

  • Click the 'Guidance' button on the web app homepage to access this manual
  • Ensure you have an internet connection to use the app and view this guide
  • Prepare your cash flow data in a CSV file with 'Date', 'Inflow', and 'Outflow' columns

Uploading Data

  • Locate the 'Upload Cash Flow Data (CSV)' section on the homepage
  • Click 'Choose File' to select your prepared CSV file
  • Click the upload area or press 'Enter' to submit the file
  • Check 'Use Sample Data' if you don’t have a CSV file to proceed with example data

Mapping Columns

  • View the 'Detected Columns' list after uploading
  • Select the correct 'Date Column' from the dropdown menu
  • Choose the 'Inflow Column' from the dropdown menu
  • Select the 'Outflow Column' from the dropdown menu
  • Review the 'Standardized Data' table to confirm mappings

Running Analysis

  • Adjust the 'Forecast Periods (Months)' slider to set the forecast length (1-24 months)
  • Move the 'Growth Rate' slider to set the expected growth (0.0-0.1)
  • Check 'Use Current 10-Year Treasury Yield' for real-time discount rate, or set it manually with the 'Discount Rate' slider (0.05-0.15)
  • Enter 'Initial Investment' amount in the number input field
  • Slide the risk factors (Policy, Intermittency, Market, Operational) to assess RARM
  • View the 'Cash Flow Forecast', 'DCF Valuation', 'RARM', and 'Sensitivity Analysis' results

Visualizations

  • Scroll to the 'Visualizations' section to see the 'Cash Flow Forecast' chart
  • Check the 'Sensitivity Analysis' heatmap below the forecast chart

Downloading Reports

  • Go to the 'Download Report' section
  • Click 'Download Report (Excel Only)' for an Excel file with forecast and sensitivity data
  • Click 'Download All Data (ZIP)' for a ZIP file with Excel data and chart images

Troubleshooting

  • If data doesn’t load, verify your CSV has 'Date', 'Inflow', and 'Outflow' columns
  • If charts don’t appear, refresh the page or check your internet connection
  • If download fails, ensure no special characters are in your CSV file
